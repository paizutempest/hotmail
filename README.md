# 🌊 DEEP OCEAN: HOTMAIL CHECKER 🛡️
> **Automated Email Validator & Account Status Checker**
> *Part of the Deep Ocean Protocol by Paizutempest*

---

## 🚀 OVERVIEW
**Hotmail Checker** adalah tools otomasi berbasis Node.js yang dirancang untuk memvalidasi daftar akun Hotmail/Outlook secara massal. Tools ini menggunakan **Deep Ocean Identity Engine** untuk memastikan setiap pengecekan dilakukan dengan footprint yang minim dan aman dari proteksi server.

Dengan tools ini, lu nggak perlu lagi cek manual satu-satu. Tinggal masukin list, duduk manis, dan biarkan si "Deep Ocean" yang bekerja bantai ribuan akun!

## ✨ FEATURES
* **High-Speed Validation**: Pengecekan status akun (Live/Die) dengan efisiensi tinggi.
* **Deep Ocean Identity Engine**: Rotasi User-Agent dan identitas browser agar tidak mudah terdeteksi sebagai bot.
* **Detailed Reporting**: Hasil pengecekan otomatis dipisahkan ke dalam file `live.txt` dan `die.txt`.
* **Multi-Thread Support**: Bisa bantai banyak akun sekaligus tanpa lemot.
* **Interactive Dashboard**: Tampilan log terminal yang berwarna-warni khas gaya Paizustore.
